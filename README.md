# Auth Backend API

A complete authentication backend built using Node.js, Express, MongoDB, JWT, and bcrypt.

## Features
- User registration
- User login
- Password hashing using bcrypt
- JWT-based authentication
- Protected profile route
- Cookie-based token handling

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt

## API Routes
- POST /api/auth/register
- POST /api/auth/login
- GET /api/auth/profile (protected)
